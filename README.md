# End-to-End Business Performance Analysis using SQL

## Overview
This project analyzes business performance using SQL by creating a relational database and extracting insights related to sales, customers, products, and regions.

## Tools Used
- PostgreSQL
- pgAdmin

## Project Structure
- Database creation (tables)
- Data insertion
- Business analysis queries

## Key Analysis
- Revenue analysis
- Customer behavior
- Product performance
- Regional performance
- Payment insights

## Key Insights
- Revenue is concentrated in a few transactions indicating early-stage growth
- Electronics products generate the highest revenue
- High-value customers contribute significantly to total sales
- Regional performance varies, indicating growth opportunities
- Failed payments indicate potential revenue leakage
- Cancelled orders highlight operational inefficiencies

## Conclusion
This project demonstrates how SQL can be used to analyze business performance and support decision-making.
